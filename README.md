# macao-venetian-hub

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的集中存放与版本管理场所。页面内容涵盖信息展示、工具页面、实验性项目等多种类型，适合开发者或爱好者参考、复用或学习。

## 目录说明

```
macao-venetian-hub/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源（如 CSS、JS、图片等）
├── docs/           # 文档或说明文件
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰，便于定位。
- `assets/`：存放页面间共享的样式、脚本或媒体资源。
- `docs/`：用于存放更详细的说明文档或开发笔记。

## 页面归档说明

本仓库中的 HTML 页面均为独立文件，可直接在浏览器中打开运行。部分页面可能依赖 `assets/` 目录下的资源，请确保文件结构完整。

页面归档遵循以下原则：

- 每个页面尽可能自包含，减少外部依赖。
- 页面命名与内容相关，方便检索。
- 历史版本通过 Git 提交记录管理，不单独保留旧版文件。

## 维护说明

- 欢迎提交 Issue 反馈问题或提出建议。
- 如需添加新页面，请在 `pages/` 下创建对应文件，并确保资源引用正确。
- 请勿提交涉及敏感内容或违反相关法规的文件。
- 本仓库主要供学习、归档和轻量级发布使用，不提供持续集成或自动部署支持。

## 许可

本仓库内容仅供学习与参考，不构成任何形式的服务或承诺。具体页面若包含第三方资源，请遵守其各自的许可协议。
